Automation using Selenium

Introduction

This project demonstrates the use of Selenium for automating interactions with a web application. Selenium is a powerful tool that allows you to simulate user actions such as clicking links, filling out forms, and navigating through pages, which is particularly useful for testing and repetitive tasks.

Automation Script Details

The script written in Python automates the following interactions on the website:

  1. Opening the application: The script begins by navigating to the main page of the application.
  2. Navigating through main pages: It demonstrates how to navigate to the "About Me" and "WebTools" pages, allowing time for each page to load and be visualized.
  3. Performing login: The script automates the login process by entering the username and password and submitting the login form.
  4. Accessing the articles section: Using the dropdown menu, the script navigates to the "Artigos" section.
  5. Creating a new article: It then clicks on the "Novo artigo" button, fills out the form with the title, content, and other required fields, and submits it.
  6. Listing and displaying the new article: Finally, the script navigates to the list of articles and scrolls to the newly created article to make it visible.
