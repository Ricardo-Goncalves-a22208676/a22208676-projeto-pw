Seletores CSS

Cascading Style Sheets (CSS) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML.

Seletor Usado Comentário
.class check_circle Seleciona elementos com a classe específica. Usado para estilizar itens específicos sem alterar outros.
#id check_circle Seleciona o elemento com o ID específico. Usado para estilizar um único elemento.
[attribute] check_circle Seleciona todos os elementos que possuem um certo atributo. Usado para formatação baseada em atributos.

Combinadores de seletores

Combinador Usado Comentário
Espaco (descendente) check_circle Seleciona todos os elementos que são descendentes do elemento base. Usado para estilizar elementos dentro de outros.
> (filho) cancel Seleciona todos os elementos filhos diretos. Usado para estilizar elementos que são diretamente dentro de outro.
